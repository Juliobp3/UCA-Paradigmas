Bienvenidos a la Semana 5

Este será el trabajo de la semana 5:

  1. Crear un archivo llamado async.js
  2. Calcular el Área de un Cuadrado: Crea una función que calcule el área de un cuadrado. La función debería aceptar la longitud del lado como parámetro y devolver el área.
  3. Calcular el Perímetro de un Rectángulo: Crea una función que calcule el perímetro de un rectángulo. La función debería aceptar la longitud y el ancho como parámetros y devolver el perímetro.
  4. Calcular el Área de un Triángulo: Crea una función que calcule el área de un triángulo. La función debería aceptar la base y la altura como parámetros y devolver el área.
  5. Calcular la Longitud de la Circunferencia: Crea una función que calcule la longitud de una circunferencia. La función debería aceptar el radio como parámetro y devolver la longitud.
  6. Calcular el Área de un Círculo: Crea una función que calcule el área de un círculo. La función debería aceptar el radio como parámetro y devolver el área.
  7. Subir el archivo a GitHub Haciendo uso de la herramienta que vimos en la clase anterior, sube el archivo a tu cuenta de GitHub. Se debe hacer uso del repositorio que creamos en la semana anterior. Crear un nuevo branch a partir del main, en este branch, vamos a agregar el nuevo archivo y cuando los cambios esten listos haran push hacia su branch(rama). Crear un pull request hacia el main. El merge sera hecho por el propietario del repositorio, en este caso Dabrin Najera.

Toma en cuenta las siguientes sugerencias:

  • Haz buen uso de los comentarios para documentar tu código.
  • Declara las variables en un solo lugar específico.
  • Organiza las funciones de operación en una sección específica del código.
  • Coloca las funciones de mostrar resultados en otra sección.
  • Al final del documento, realiza los llamados necesarios para ejecutar cada función.
